# 开发指南

## 开发环境搭建

### 1. 克隆项目

```bash
git clone <repository-url>
cd 深度学习模型
```

### 2. 后端开发环境

#### 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate  # Windows
```

#### 安装依赖

```bash
pip install -r requirements.txt
```

#### 安装开发工具

```bash
pip install pytest pytest-cov black flake8 mypy
```

### 3. 前端开发环境

```bash
cd frontend
npm install
```

#### 安装开发工具

```bash
npm install -D @vue/test-utils vitest eslint prettier
```

```python
    tiles: List[Dict],
    process_func: callable,
    num_workers: int = 8
) -> List[Dict]:
    """并行处理瓦片

    使用多进程并行处理影像瓦片，提高处理效率。

    Args:
        tiles: 瓦片列表，每个瓦片包含数据和元信息

        处理结果列表，与输入瓦片顺序对应

    Raises:
        ValueError: 如果tiles为空或process_func不可调用
        RuntimeError: 如果处理过程中发生错误

    Example:
        >>> tiles = split_image_into_tiles(image)
        >>> results = parallel_process_tiles(tiles, process_func)
    """
    pass
```

## 性能优化

### 后端优化

#### 1. 数据库查询优化

```python
# 使用索引
class Task(Base):
    __tablename__ = "tasks"

    id = Column(String, primary_key=True)
    status = Column(String, index=True)  # 添加索引
    created_at = Column(DateTime, index=True)

# 批量查询
tasks = session.query(Task).filter(
    Task.status.in_(['pending', 'running'])
).all()

# 使用分页
tasks = session.query(Task).limit(20).offset(page * 20).all()
```

#### 2. 缓存优化

```python
from functools import lru_cache

@lru_cache(maxsize=128)
def load_model(model_path: str):
    """加载模型（带缓存）"""
    return torch.load(model_path)
```

#### 3. 并行处理优化

```python
# 使用进程池
from multiprocessing import Pool

def parallel_process(items, num_workers=8):
    with Pool(num_workers) as pool:
        results = pool.map(process_item, items)
    return results
```

#### 2. 防抖和节流

```typescript
// 防抖
function debounce<T extends (...args: any[]) => any>(
  func: T,
  wait: number
): (...args: Parameters<T>) => void {
  let timeout: NodeJS.Timeout;
  return function(...args: Parameters<T>) {
    clearTimeout(timeout);
    timeout = setTimeout(() => func(...args), wait);
  };
}

// 使用
const handleSearch = debounce((query: string) => {
  // 搜索逻辑
}, 300);
```

## 常见问题

### 1. GDAL导入错误

```bash
# 确保GDAL已安装
brew install gdal  # macOS
sudo apt-get install gdal-bin libgdal-dev  # Ubuntu

# 设置环境变量
export GDAL_DATA=/usr/local/share/gdal
```

### 2. 端口冲突

```bash
# 查找占用端口的进程
lsof -i :8000

# 杀死进程
kill -9 <PID>
```

### 3. 依赖冲突

```bash
# 清理pip缓存
pip cache purge

# 重新安装依赖
pip install -r requirements.txt --force-reinstall
```

### 4. 前端构建失败

```bash
# 清理node_modules
rm -rf node_modules package-lock.json

# 重新安装
npm install
```
