# 功能使用指南

## 1. 无监督检测

无监督检测是最简单快速的使用方式，无需训练样本即可开始检测。

### 1.1 准备影像

**支持的格式**:
- GeoTIFF (.tif, .tiff)
- ERDAS IMAGINE (.img)

**影像要求**:
- 多光谱遥感影像（至少3个波段）
- 包含地理坐标信息
- 建议分辨率：0.5-2米

### 1.2 上传影像

1. 访问前端界面 http://localhost:5173
2. 点击左侧菜单"无监督检测"
3. 点击"选择文件"按钮
4. 选择待检测的遥感影像
5. 等待上传完成

**大文件上传**:
- 系统自动分片上传（5MB/片）
- 支持断点续传
- 显示实时上传进度
- 上传失败可重试

### 1.3 设置检测参数

**聚类数量 (n_clusters)**:
- 默认值：5
- 范围：2-10
- 说明：将影像分为几类，通常设置为5-7
- 建议：先用默认值尝试，根据结果调整

**最小面积 (min_area)**:
- 默认值：100平方米
- 范围：10-1000
- 说明：过滤小于此面积的斑块
- 建议：根据实际需求调整，避免过多碎斑

### 1.4 启动检测

1. 确认参数设置
2. 点击"开始检测"按钮
3. 系统创建检测任务
4. 自动跳转到任务进度页面

### 1.5 查看进度

任务进度页面实时显示：
- 当前状态（运行中/已完成/失败）
- 完成百分比
- 当前处理步骤
- 预计剩余时间（参考）
- 可以随时取消任务

### 1.6 查看结果

检测完成后显示：
- **地图展示**：检测结果叠加在影像上
- **统计信息**：
  - 总面积
  - 病害木面积
  - 病害率
  - 斑块数量
- **GeoJSON下载**：可下载结果用于GIS分析
- **结果修正**：可手动修正检测结果

### 1.7 结果修正

如果检测结果不够准确，可以手动修正：

1. 点击"修正结果"按钮
2. 在地图上：
   - 点击斑块可删除
   - 绘制新的斑块
   - 调整斑块边界
3. 保存修正后的结果
4. 重新生成统计信息

## 2. 监督检测

监督检测基于深度学习模型，精度更高，但需要先准备训练样本和训练模型。

### 2.1 准备训练样本

#### 样本要求

**病害木样本**:
- 包含明显病害特征的影像
- 尺寸：建议256×256像素以上
- 数量：至少100个样本
- 标签：disease

**健康木样本**:
- 健康森林的影像
- 尺寸：与病害木样本一致
- 数量：与病害木样本数量相当
- 标签：healthy

#### 样本制作

可以使用以下工具制作样本：
- QGIS：裁剪影像
- ArcGIS：提取样本
- Python脚本：批量处理

### 2.2 上传训练样本

1. 点击"训练样本管理"菜单
2. 点击"上传样本"按钮
3. 选择样本文件（支持批量上传）
4. 选择样本标签（disease/healthy）
5. 添加样本描述（可选）
6. 点击"确认上传"

### 2.3 样本管理

**查看样本**:
- 样本列表显示所有已上传样本
- 可按标签筛选
- 显示样本缩略图
- 显示样本信息

**删除样本**:
- 选择要删除的样本
- 点击"删除"按钮
- 确认删除

**样本统计**:
- 病害木样本数量
- 健康木样本数量
- 样本总数
- 样本分布

### 2.4 配置训练参数

1. 点击"模型训练"菜单
2. 设置训练参数：

**模型名称**:
- 自定义模型名称
- 建议包含日期和主干网络信息
- 例如：VGG16_2024-01-01

**主干网络**:
- VGG11：轻量级，训练快
- VGG16：经典网络，效果好
- VGG19：更深层次，精度高
- ResNet18：残差网络，训练稳定
- ResNet34：更深的残差网络
- ResNet50：深度残差网络，精度最高

**训练轮数 (epochs)**:
- 默认值：100
- 范围：10-500
- 说明：训练的迭代次数
- 建议：先用100轮，观察收敛情况

**批次大小 (batch_size)**:
- 默认值：32
- 范围：8-128
- 说明：每次训练使用的样本数
- 建议：根据显存大小调整，显存小用16

**学习率 (learning_rate)**:
- 默认值：0.001
- 范围：0.0001-0.01
- 说明：模型参数更新步长
- 建议：使用默认值，训练不收敛时降低

### 2.5 启动训练

1. 确认参数设置
2. 点击"开始训练"按钮
3. 系统创建训练任务
4. 跳转到训练进度页面

### 2.6 监控训练

训练进度页面显示：
- **当前轮数**：已完成的训练轮数
- **训练损失**：训练集上的损失值
- **训练准确率**：训练集上的准确率
- **验证损失**：验证集上的损失值
- **验证准确率**：验证集上的准确率
- **损失曲线**：实时绘制损失变化
- **准确率曲线**：实时绘制准确率变化

**判断训练效果**:
- 损失值持续下降：训练正常
- 准确率持续上升：模型在学习
- 验证准确率不再提升：可能过拟合
- 损失值不下降：学习率可能过大

### 2.7 使用模型检测

训练完成后，可以使用模型进行检测：

1. 点击"监督检测"菜单
2. 选择训练好的模型
3. 上传待检测影像
4. 设置置信度阈值（默认0.8）
5. 点击"开始检测"
6. 等待检测完成
7. 查看检测结果

**置信度阈值**:
- 范围：0.5-0.95
- 说明：只保留置信度高于此值的检测结果
- 建议：0.8是较好的平衡点
- 阈值高：结果更可靠，但可能漏检
- 阈值低：检测更全面，但可能误检

## 3. 大文件上传

系统支持大文件分片上传，适用于GB级别的遥感影像。

### 3.1 上传流程

1. 选择文件后，系统自动分片（5MB/片）
2. 并发上传多个分片（默认2个并发）
3. 实时显示上传进度和速度
4. 所有分片上传完成后自动合并
5. 验证文件完整性
6. 返回文件路径

### 3.2 断点续传

如果上传中断：
1. 系统记录已上传的分片
2. 重新上传时自动跳过已上传分片
3. 只上传未完成的分片
4. 节省时间和带宽

### 3.3 上传优化

**提升上传速度**:
- 使用有线网络
- 关闭其他占用带宽的程序
- 选择网络状况好的时段

**处理上传失败**:
- 检查网络连接
- 检查磁盘空间
- 重试上传
- 查看错误日志

## 4. 结果管理

### 4.1 查看历史任务

1. 点击"历史任务"菜单
2. 查看所有历史任务列表
3. 可按类型、状态筛选
4. 点击任务查看详情

### 4.2 下载结果

**GeoJSON格式**:
- 包含检测到的所有斑块
- 包含地理坐标信息
- 可在QGIS、ArcGIS中打开
- 可用于进一步分析

**统计报告**:
- 检测统计信息
- 可导出为CSV或Excel
- 包含面积、数量等指标

### 4.3 结果对比

可以对比不同检测结果：
1. 选择两个或多个任务
2. 点击"对比"按钮
3. 并排显示检测结果
4. 对比统计指标
5. 分析差异原因

## 5. 任务管理

### 5.1 查看任务状态

实时查看任务状态：
- 等待中：任务已创建，等待执行
- 运行中：任务正在执行
- 已完成：任务成功完成
- 失败：任务执行失败
- 已取消：任务被用户取消

### 5.2 取消任务

如果需要取消正在运行的任务：
1. 进入任务详情页面
2. 点击"取消任务"按钮
3. 确认取消
4. 系统优雅终止任务
5. 清理临时文件

### 5.3 重试任务

如果任务失败，可以重试：
1. 查看失败原因
2. 修正问题（如参数错误）
3. 点击"重试"按钮
4. 使用相同参数重新执行

### 5.4 删除任务

删除不需要的任务：
1. 选择要删除的任务
2. 点击"删除"按钮
3. 确认删除
4. 系统删除任务记录和结果文件

## 6. 常见问题

### 6.1 检测结果不准确

**无监督检测**:
- 调整聚类数量
- 调整最小面积阈值
- 尝试不同的影像预处理方法
- 使用监督检测获得更高精度

**监督检测**:
- 增加训练样本数量
- 提高样本质量
- 调整训练参数
- 尝试不同的主干网络
- 延长训练时间

### 6.2 处理速度慢

- 减少并行处理数量
- 使用更小的影像进行测试
- 升级硬件配置
- 使用GPU加速（如果支持）

### 6.3 内存不足

- 减少批次大小
- 减少并行处理数量
- 关闭其他占用内存的程序
- 增加系统内存

### 6.4 上传失败

- 检查文件格式是否支持
- 检查文件是否损坏
- 检查磁盘空间是否充足
- 检查网络连接是否稳定
- 查看错误日志获取详细信息

## 7. 最佳实践

### 7.1 影像准备

1. **预处理**：进行辐射校正、大气校正
2. **裁剪**：裁剪到研究区域，减少处理时间
3. **格式**：使用GeoTIFF格式，兼容性好
4. **压缩**：使用LZW压缩，减小文件大小

### 7.2 样本制作

1. **数量**：每类至少100个样本
2. **质量**：选择典型、清晰的样本
3. **平衡**：各类样本数量相当
4. **多样性**：覆盖不同季节、不同程度的病害

### 7.3 模型训练

1. **从简单开始**：先用VGG16训练
2. **观察收敛**：监控损失和准确率曲线
3. **防止过拟合**：验证准确率不再提升时停止
4. **保存最佳模型**：保存验证准确率最高的模型

### 7.4 检测应用

1. **先小范围测试**：用小影像测试效果
2. **调整阈值**：根据测试结果调整置信度阈值
3. **人工复核**：对检测结果进行人工复核
4. **持续优化**：根据实际应用效果优化模型
