## 已知问题和改进建议

### 问题1: 大量小分片测试失败
**现象**: 5MB 文件分成 50 个 100KB 分片时失败

**原因分析**:
- 可能是速率限制导致部分请求被拒绝
- 或者是快速连续请求时的并发问题

**改进建议**:
- 客户端应实现指数退避重试
- 增加请求间隔
- 或者调整服务器端的速率限制配置

### 问题2: 无效文件大小验证
**现象**: fileSize=-1 的请求被接受 (200 OK)

**原因分析**:
- 后端可能没有验证负数的文件大小

**改进建议**:
- 在 Pydantic 模型中添加 gt=0 验证
- 或者在上传处理中添加额外的验证

### 问题3: 不完整分片处理测试
**现象**: 会话不存在返回 404

**原因分析**:
- 测试使用了不同的上传 ID
- 系统正确地拒绝了不存在的会话

**改进建议**:
- 测试脚本应使用相同的上传 ID
- 或者系统应返回更清晰的错误信息
